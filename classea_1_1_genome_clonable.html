ea::GenomeClonable< T > Class Template Reference

Detailed Description

template<class T>
class ea::GenomeClonable< T >

The specialization of Genome to provide default shallow-clone method.

This class is designed by using the CRTP. Therefore, users should inherit this class by providing the child class as the template argument. For example:

class A : public GenomeClonable<A> { ... }

Users must notice that if the representation requires deep-copy, this class must not be inherited. Such classes must inherit directly from Genome and override a version of Genome::CloneBase() for themselves.

Template Parameters
TThe child class which inherits GenomeClonable.
