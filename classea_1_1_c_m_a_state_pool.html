ea::CMAStatePool Class Reference

Detailed Description

Extension of Pool to store the state of the CMAEvolutionStrategy algorithm.

Because a Population can only store a Pool, states of CMAEvolutionStrategy must be kept in a Pool in order to be saved and restored. The states of a CMA-ES include:

  • N: The number of dimensions \(N\)
  • mean: The current mean \(m\)
  • ps: The isotropic evolution path \(p_\sigma\)
  • pc: The anisotropic evolution path \(p_c\)
  • sigma: The step-size \(\sigma\)
  • B: The eigenvectors of covariance matrix \(C\)
  • D: The singular values of covariance matrix \(C\)

The covariance matrix \(C\) can be restored from B and D by using the function GetC(). The square root of inverse covariance matrix \(C^{-1/2}\) could also be calculated by using GetInvSqrtC().
