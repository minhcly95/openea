ea::Organism Class Referencefinal

Detailed Description

Wrapper of Genome with its corresponding Fitness.

Organism class represents a Genome has been evaluated. It is the data structure mainly used after evaluation phase and stored after each generation. It also provides Serialize() and Deserialize() functions to enable back-up feature (see BackupHook).

See also
Population::Evolve()

Implementation of deserialization method.

Child classes should override this function to implement their own deserialization method. The deserialization method should match with the serialization one.
