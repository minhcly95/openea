ea::Genome Class Referenceabstract

Detailed Description

The interface for representation.

This class provides the interface to implement custom representation. It provides a pure virtual method to clone the Genome (see CloneBase()).

However, if the custom representation doesn't require deep copy, users should inherit GenomeClonable instead to have a default shallow-clone function. Otherwise, users must write for themselves a cloning method.
