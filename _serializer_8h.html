Define a specialization of Serializer.

This macro will specialize ea::Serializer::TypeName() by using the string form of TYPE and declare the signature of ea::Serializer::DoParse(). Users should follow this macro by the brackets to define the implementation.

Example of specialization of ea::Serializer for type string :

1 EA_DEFINE_CUSTOM_SERIALIZER(string, data, ss) {
2  int length = ss.rdbuf()->in_avail() + 1;
3  char* buffer = new char[length];
4  ss.read(buffer, length);
5  buffer[length - 1] = '\0';
6  data = string(buffer);
7  delete[] buffer;
8  return true;
9 }
