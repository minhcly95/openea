ea::EAMLReader Class Reference

Detailed Description

Reader for the EAML file format.

This class provides a set of functions to load an EAML configuration file and reconstruct a Population from that file. To use the reader, simply create the reader by calling the constructor EAMLReader() with the file name and then call Build() to reconstruct the Population.

StrategyPtr ea::EAMLReader::Build ( const VariableMap &  map)

Reconstruct a Population from the EAML file with the given VariableMap.

This function uses the NameService and TypeInfo routines to reconstruct whole a Population only from configuration file. If the file doesn't follow the format or misses some operators, exception will be thrown.

The VariableMap will contain a map of Variable keys and new values. These values will be used instead of the default values defined in the EAML file.

Parameters
mapThe VariableMap containing pairs of key and value.
Returns
The reconstructed Population.
