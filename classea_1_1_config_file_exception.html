ea::ConfigFileException Class Reference

Detailed Description

Specialization of EAException for the EAMLReader.

Logical exceptions thrown in the EAMLReader are mostly ConfigFileException. This class provides a new bunch of ConfigFileException::ErrorCode which is useful in the context of reading XML file.
