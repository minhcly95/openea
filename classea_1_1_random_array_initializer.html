ea::RandomArrayInitializer< T > Class Template Reference

Detailed Description

template<class T>
class ea::RandomArrayInitializer< T >

A random Initializer for ArrayGenome.

This Initializer will create Genome of type ArrayGenome with random content for a given array length. The Initializer accepts a Randomizer to generate random content (it doesn't require for BoolRandomArrayInitializer).

Template Parameters
TType of genes.
EAML Name
BoolRandomArrayInitializer, IntRandomArrayInitializer, DoubleRandomArrayInitializer
EAML Attributes
length
uint - Required - The length of generated ArrayGenome.
randomizer
Randomizer - Required (Optional for BoolRandomArrayInitializer) - The random value generator.
