ea::PointResetMutation< T > Class Template Reference

Detailed Description

template<class T>
class ea::PointResetMutation< T >

Implementation of Point reset mutation operator for ArrayGenome.

This Mutator is only applicable on ArrayGenome of the same type of genes T.

When apply, each gene in the ArrayGenome will have a chance to be reset or flipped (i.e. take a new random value). Users can configure whether the new gene is allowed to have the same value with the old one.

Note
The name used for PointResetMutation of gene type bool in NameService is FlipBitMutation instead of BoolPointResetMutation
Template Parameters
TType of genes.
Method
Iterate through the array of genes, then apply probability test.
Time Complexity
O(N)
EAML Name
FlipBitMutation, IntPointResetMutation, DoublePointResetMutation
EAML Attributes
rate
double - Required - The probability that each gene will be reset with.
randomizer
Randomizer - Required (Optional for FlipBitMutation) - The random value generator.
allow-dupplicate
bool - Optional (Not exist for FlipBitMutation) - Whether the new gene is allowed to have the same value with the old one.
