void ea::Phenome::DoDeserialize ( istream &  pStream)
overrideprotectedvirtual

Implementation of deserialization method.

Child classes should override this function to implement their own deserialization method. The deserialization method should match with the serialization one. If this is not overriden, then default behaviour is doing nothing (that means this function doesn't need to be overriden for classes which don't require saving any data except for class identity).
