virtual VectorPtr<GenomePtr> SphereInitializer::DoInitialize ( uint  pSize)
inlineoverrideprotectedvirtual

Implementation of the initialization method (need to be overriden in child class).

The method requires a pool size as the input parameter. The implementation of this method should return a vector of Genome with the vector size equal to the input pool size. Otherwise, the proxy caller Initialize() will throw an error for inconsistent implementation.

Parameters
pSizeThe expected size of the returned vector.
Returns
A vector of Genome with size equals to the expected size.
