ea::NameService Class Reference

Detailed Description

Static class which provides a service to translate class name into TypeInfo.

This class is the entry point to construct any object during run-time. It provides a dictionary to translate class name of type string to TypeInfo, and from there a real object can be constructed. All functions of this class are static, so the NameService can be accessed everywhere.

Before translating any name, users must add the class to the NameService first by using Add() function. Then simply calling Get(), users can get the TypeInfo of a given class name. Built-in types in the library are added by default for convenience.
