Detailed Description

template<class T>
class ea::TypedRecombinator< T >

A Recombinator which requires the input and output Genome of type T instead of Genome.

It is usually the case that users know which type of Genome they are processing. Therefore, to avoid putting too much responsibility of type-casting on users, this class will automatically cast the input and output Genome of Recombinator::DoCombine(vector<GenomePtr>&) into genomes of type T. Users will override the new function with the same name DoCombine(vector<Ptr<T>>&) instead of the old one.

Template Parameters
TMust be a child of Genome.

virtual Ptr<T> ea::TypedRecombinator< T >::DoCombine ( vector< Ptr< T >> &  pParents)
protectedpure virtual

Implementation of the recombination operator with input and output of type T.

This is the new implementation point of the recombination operator. The input will be auto-casted to type T and the function only requires a vector of Ptr<T>. Moreover, users only need to return an output of type T also, and it will be converted back into Genome automatically to be compatible with Recombinator.

The implementation must still follow the rule described in Recombinator::DoCombine(vector<GenomePtr>&).

Parameters
pParentsThe list of parents of type T which need to be recombined.
Returns
A completely new genome of type T object which is the combined result of the input parents.

Specialized implementation to be compatible with Recombinator.

This function will cast the input, call DoCombine(vector<Ptr<T>>&) and convert back the output internally. This function also provides an informative error message if bad cast happened.
