ea::ClusterComputableBase Class Referenceabstract

Detailed Description

Base class for all ClusterComputable template.

This class provides a way to generalize ClusterComputable template classes, so Cluster class can process them using inheritance polymorphism instead of templating one. This class also provides cluster operator ID management function such as GetClusterOpId().
