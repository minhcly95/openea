OpenEA Framework Documentation

Introduction

Welcome to the website of the OpenEA project. The OpenEA project is an open-source framework for Evolutionary Computing built in C++. The framework is general-purpose and can be expanded to fit any demand without sacrificing speed and reusability. Here are the main features of the framework:

  • General interface for representations and fitness: Genome, Fitness, Pool
  • General interface for operators: Variator, Selector, Evaluator
  • General interface for algorithms: Strategy, Operator
  • Two programming options: C++ code or EAML configuration file
  • Powerful Command-Line Interface (CLI) allowing fast configuration
  • Plugin loader which loads user code without recompiling
  • Parallelism via MultiThreading or Distributed Computatation on Cluster
  • Back-up to and Restore from file system
  • Speed tracker (both generational and for each phase)

Installation

Currently, the framework is only tested on Ubuntu 16.04 system. There are two ways to install the framework: from Ubuntu Launchpad (apt-get) and directly from GitHub.

Ubuntu Launchpad

To install the framework using apt-get, first, add the following PPA to your system:

sudo add-apt-repository ppa:minhcly95/openea
sudo apt-get update

Then, install the package libopenea-cli which includes both the library and the CLI tool:

sudo apt-get install libopenea-cli

To test the installation, enter the command:

openea

And a list of available sub-commands should be printed on the terminal.

GitHub

To install from GitHub, first, clone the repository to your computer:

git clone https://github.com/minhcly95/openea.git

Then, run the Makefile using the command:

sudo make install

This command will compile the source and create symbolic links to the binary and source files, so you can include and link them any where on your machine. To test the installation, enter the command:

openea

And a list of available sub-commands should be printed on the terminal.

How to use

For new users, it is recommended to read the Tutorials page to get used to the framework. More tutorials are being written, so don't be disappointed. All available operators are described in the Built-in Classes page. For more information, check the API documentation of each class.

Note
Users of the library should understand the basic operation of Evolutionary Computing and C++ programming. If you don't know what is an Evolutionary Algoithm, I recommend the book Introduction to Evolutionary Computing, 2nd Edition, 2015 by A.E. Eiben and J.E. Smith.
