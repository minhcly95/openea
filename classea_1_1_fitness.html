ea::Fitness Class Referenceabstract

Detailed Description

The interface for general fitness.

This class only serves as the base class of fitness specialization. It also defines a comparison method between Fitness object (see Compare() function) and useful overloaded operators.

For implementation of Fitness for single scalar fitness value, see ScalarFitness.
