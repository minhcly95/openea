ea::UnifiedData Class Reference

Detailed Description

Unified data structure to store both string and Constructible object type.

Value of an attribute could be simple as a string or be complex as another Constructible object. Therefore, an unified data structure will encapsulate the difference of nature and provide a simple view to the data. To create an unified data struct, simply construct it with a string, a Constructible object or a list of Constructible.
